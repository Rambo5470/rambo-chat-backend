## BATTERY / CHARGING ISSUES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SYMPTOM: Battery won't charge at all
→ Step 1: Check charger connection (both ends), try different outlet
→ Step 2: Check charge port on bike for damage/debris
→ Step 3: Leave charger connected 30 min — some BMS need time to wake up
→ Step 4: If still no charge → replacement battery (use battery selector: rambobikes.com/pages/battery-selector)
→ Replacement charger: RP-11-12-01 (Trailbreaker/Chameleon) or ask for model to confirm

SYMPTOM: Battery drains very fast / not holding charge
→ Check if stored in cold temperatures (cold kills capacity temporarily)
→ Trailbreaker: RP-10-17-02 battery replacement most common fix
→ If under 2 years old and drains in <1 hour → escalate to Misti

SYMPTOM: Want to upgrade battery
→ Krusader: Cannot upgrade beyond 14Ah from factory
→ Hellcat: R127-30 ($1,349.99) OR Dual Battery Kit — offer both
→ All others: rambobikes.com/pages/battery-selector — do NOT guess compatibility

## MOTOR / NO POWER / WON'T TURN ON
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SYMPTOM: Bike won't turn on at all
→ AWD bikes (Krusader/Megatron/Hellcat): Power button is on BOTTOM of display — check underneath first
→ All bikes: Ensure battery fully seated and locked in, fully charged
→ Try holding power 3-5 seconds (not a quick tap)
→ If still nothing → display replacement most common first fix (not motor)

SYMPTOM: Motor cuts out / intermittent power
→ Check all cable connections at motor, controller, battery
→ Power off 30 seconds, restart
→ If error code shows → see ERROR CODES section
→ Persistent: display + throttle replacement (sent together in past cases)

SYMPTOM: No pedal assist / throttle not working
→ Throttle connector: check at handlebar (near grip) — NOT near motor
→ If throttle doesn't respond at all → RP-04-23-01 (Trailbreaker throttle)
→ Mid-drive bikes (Dominator/Rebel/Savage G3): confirm cadence sensor working

## ERROR CODES — PROVEN FIXES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

E03 (Brake engaged): Check both brake levers are fully released. Brake sensor stuck.
→ Past fix: Brake lever sensor adjustment or replacement

E04/E05 (Throttle): Debris near throttle grip. Check connector at handlebar.
→ Past fix: Clean throttle grip area, reseat connector

E06 (Low voltage): Battery low or BMS fault. Charge fully first.
→ If persists after full charge → battery replacement

E08 (Hall signal fault): Motor sensor issue.
→ Past fix: motor replacement — escalate to Misti for parts order

E21 (Speed sensor): MOST COMMON ERROR.
→ Fix video (share this link): https://youtu.be/snKZ0jPSVHU
→ Past cases: 95% resolved with this video alone

E22 (BMS fault): Battery management system issue.
→ Past fix: battery replacement — escalate to Misti

E30 (Communication fault): Display/controller communication broken.
→ Past fix: sent display AND controller together (not one or the other)
→ Check all cable connections first

## DISPLAY / SCREEN ISSUES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SYMPTOM: Display won't power on / blank screen
→ First: check power button location (AWD = bottom of panel)
→ Check battery connection
→ Past fix: replacement display (most common resolution in 300+ cases)

SYMPTOM: Display powers on but shows wrong info / errors
→ Try factory reset: hold M button 5+ seconds on most models
→ Dominator: check class setting (double-tap power)
→ E30 communication error → display + controller together

SYMPTOM: Dominator UD light issues
→ Customer needs to locate the toggle switch — common question
→ Past cases: ordered replacement light and LCD screw and toggle switch

## DRIVETRAIN (CHAIN / GEARS / DERAILLEUR)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SYMPTOM: Squeaky chain / drivetrain noise
→ Past fix: Oil/lube the chain — this resolves majority of squeaky cases
→ Product: any bike chain lube, NOT WD-40

SYMPTOM: Chain skipping / not shifting
→ Check chain tension first (tighten if loose)
→ Mid-drive bikes: chain wear is FASTER than hub drive — normal
→ Replacement chain: RP-15-03 → rambobikes.com/products/replacement-bike-chain
→ Always recommend derailleur hanger check: RP-23-02 ($29.99) when derailleur involved

SYMPTOM: Derailleur bent / damaged
→ Always recommend RP-23-02 (hanger) alongside derailleur replacement
→ Rebel 2.0: ask for SERVICE CODE on back of seat tube — critical for correct parts

SYMPTOM: Gear indexing off
→ Barrel adjuster fix (turn barrel adjuster on derailleur 1/4 turn at a time)
→ If bent derailleur hanger → local bike shop recommended

## WHEELS / TIRES / TUBES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

SYMPTOM: Flat tire / noise from wheel
→ Past case insight: tube flat was cause of wheel noise in many cases
→ 24×4.0 tube (most fat tire): RP-09-01 → rambobikes.com/products/bike-tire-tubes
→ 26×4.0 (Savage G3 only): RP-09-01 (select 26×4.0)
→ R750 G3 rear wheel: RP-07-01-14 (3-speed $79.99) or RP-07-08 (single speed $169.99)
→ Bushwacker rear wheel: ask 3-speed OR 5-speed Sturmey Archer hub first

## SHIPPING DAMAGE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

PROCESS (from hundreds of past cases):
1. Ask customer to send photos of damage + original box
2. Document the specific damaged parts
3. Escalate to Misti with photos — she approves replacement parts
4. Replacement parts ordered (not full bike replacement in most cases)
5. Common shipping damage items: quick release, derailleur, rack hardware, fork

IMPORTANT: Customer MUST have registered their bike OR provide purchase receipt.
If unregistered: ask them to register first at rambobikes.com/pages/product-registration

## PRE-SALES / MODEL RECOMMENDATIONS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Most common pre-sales questions from 13,000+ cases:

"Which bike is best for hills?"
→ Mid-drive (Dominator/Rebel) for steep hills — motor in center = better torque
→ AWD (Krusader/Megatron) for all-terrain including hills

"Best bike for larger riders (6'4"+)?"
→ Suggest 26" wheel bikes: Hellcat, Megatron, Dominator — NOT Krusader (24" tires)

"Kids bike options?"
→ Trailbreaker 3.0 (recommended first) or Chameleon
→ Lil Whip: DISCONTINUED — do not suggest

"Megatron vs Krusader?"
→ Megatron: 26" wheels, 2×1000W, better for taller riders, heavier terrain
→ Krusader: 24" tires, 2×500W, lighter, good all-around
→ Megatron has NO Bluetooth/app — display only

"Can I ride on road?"
→ All Rambo bikes are street legal as Class 1/2/3 (varies by model and state)
→ Direct to class settings instructions for their model

## REGISTRATION PATTERNS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Most common issues (from 310 registration cases):
1. Serial number not found → direct to head tube AND controller box AND original box
2. Purchase receipt missing → ask customer to provide
3. Wrong bike model entered → clarify which model before completing
4. Dealer didn't register for them → common, handle it via rambobikes.com/pages/product-registration

Valid registration: Serial starts with 2-4 letters + 6-8 numbers
Invalid format examples: all numbers, all letters, too short — ask for photo of head tube
